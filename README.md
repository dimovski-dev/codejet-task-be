## Run Backend Locally

Clone the project

```bash
  git clone https://github.com/dimovski-dev/codejet-task-be
```

Go to the project directory

```bash
  cd codejet-task-be
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  docker-compose up -d
```


## Run Simple Frontend Locally

Clone the project

```bash
  open index.html
```


## Running Tests

To run tests, run the following command

```bash
  yarn test
```
